# Draft Club — движок (Фаза 2a, headless)

Чистая логика драфта без UI/сети/БД. Авторитарный движок: валидирует ходы,
сам считает автопасс, ведёт очередь, фазу замен и подсчёт. Переиспользуется
будущим веб-сервером (Node) напрямую. Node v24, ESM, без сборки.

Правила: `Personal/Draft Club/{draft-club} {spec} правила драфта` (в vault).

## Файлы
- `config.js` — параметры (бюджет, позиции, лимит клуба, шаг ставки).
- `rules.js` — `canAdd` (физибилити + позиции + лимит клуба), `maxBid`.
- `draft.js` — state-machine: lobby → nominating/bidding → substitutes → done.
- `scoring.js` — итоги: очки тренера, срабатывание замены, тай-брейки.
- `data.js` — синтетический пул (в проде заменится игроками FanTeam + тренерами).
- `sim.js` — прогон N случайных драфтов ботами + проверка инвариантов.
- `scoring.test.js` — детерминированные тесты подсчёта.

## Запуск
```
node engine/sim.js 500          # 500 драфтов, проверка правил
node engine/scoring.test.js     # тесты подсчёта
```

## Статус
- Аукцион: 500/500 валидных драфтов (составы 12, GK=1/COACH=1, позиции в рамках,
  бюджет ≥0, лимит клуба ≤5, без дублей, у всех замена).
- Подсчёт: 5/5 тестов.
